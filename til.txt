1. 항해99 프로젝트들
2. 1주차 주요 개념 키워드

- 버전관리 : 변경된 상태를 정리하는 것
- commit : 변경된 프로젝트 상태를 저장  ( 1일 1커밋, 하루에 하나씩 저장 -사용자,시간,메시지,아이디- )

커밋을 하려면
1. git initialize ( 깃 초기화 )
컴퓨터 폴더를 Git 에서 관리하는 프로젝트로 지정  
2. add ( staging )
커밋할 파일을 선택하는 것
3. 메세지 입력 후 commit

- repo, repository 리포지토리 'Git으로 관리되는 프로젝트'
내 컴퓨터 = 로컬 repo
클라우드 = 원격 repo
- Tracking, 추적
로컬 repo와 원격 repo를 연결한다

- 로컬 repo -> 원격 repo = Push, 밀어
- 원격 repo - > 로컬 repo = Pull, 땡겨
- Clone 
원격 repo를 내 컴퓨터에 가져와서 초기 repo 세팅하는 것 ( 복제하기 )
